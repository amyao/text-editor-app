# Collaborative Text Editor

A real-time collaborative rich-text editor built as a vibe coding assessment.

## Planned features

- Rich-text input and display
- Bold, italic, text color, font size, lists, and paragraphs
- Clear editor content
- Save content to local storage
- Live word count
- Multi-user collaborative editing with cursor presence
- Version history and revision archive
- Review workflow with persistent text highlights
- Inline comments similar to Microsoft Word and Google Docs
